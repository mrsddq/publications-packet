# ClipCap

**Authors**: Mokady, Hertz, Bermano
**Year**: 2021
**Link**: https://arxiv.org/abs/2111.09734
**Implementation**: https://github.com/mrsddq/clip-image-captioning

## Key Contribution
ClipCap maps CLIP embeddings into a language-model prefix for image captioning.

## What I Implemented / Adapted
The repository uses the same central idea of frozen CLIP visual features feeding a trainable language decoder.
